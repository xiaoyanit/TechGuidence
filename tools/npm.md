#NPM

##Commands

+ npm init [-f]
  create package.json
 
+ npm i [xxx] --save  
  save dependencies to package.json
  
+ npm i [xxx] --save-dev 
  save devDependencies to package.json

+ npm install  
  install all dependencies in package.json
  
+ npm install --production
  install dependencies in package.json except devDependencies
